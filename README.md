Advanced Blog Application

A simple blog application built with Next.js.

The application also includes client-side and server-side rendering (SSR) for optimized performance. Users can search for blog posts, leave comments, and view the full content of individual posts. The application is styled with modern CSS, and includes client-side form validation to ensure data integrity.

## Features

    Paginated list of blog posts with titles and excerpts
    Individual post pages with full content and comments
    Create new post page with form
    Server-side rendering (SSR) for home and post pages
    Search functionality by title
    Client-side form validation for create post and comment forms
    Basic CSS styling

## Getting Started

    Clone this repository:
    git clone https://github.com/sameera222/advanced-blog-application.git

Install the dependencies:

npm install

Run the development server:
npm run dev

Open the browser and go to http://localhost:3000.
